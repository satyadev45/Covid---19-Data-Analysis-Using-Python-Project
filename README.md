# Covid---19-Data-Analysis-Using-Python-Project:

üìä Project Overview:

This Data Analysis Project analyzes global COVID-19 statistics across 209 countries using Python Data Science Stack. The analysis covers data cleaning, exploratory data analysis (EDA), global KPI calculations, and creates professional visualizations including bar charts,  pie charts, scatter plots, bubble charts, and correlation heatmaps.

Key Business Questions Answered:
- Which countries have highest Total Cases and Total Deaths.
- What are Continent-wise case distributions.
- What is the Global Death Rate and Recovery Rate.
- How do Cases vs Deaths correlate across regions?
